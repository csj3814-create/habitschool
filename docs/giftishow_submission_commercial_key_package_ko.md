# 해빛스쿨 Giftishow 상용 Key 승인 제출 문서

## 서비스 개요

- 서비스명: 해빛스쿨(Habit School)
- 운영사: 공감케어
- 서비스 성격: 건강 습관 형성 보상 서비스
- 쿠폰 교환 자산: 포인트
- 현재 운영 정책: 포인트 차감 후 Giftishow API 연동 쿠폰 발급
- 쿠폰 수령 방식: 앱 내 쿠폰 보관함 우선 노출
- 관리자 운영 방식: 관제탑에서 발급 상태, 예외 건, 재조회 이력 관리

## 서비스 이용 흐름

1. 회원이 해빛스쿨 앱에서 건강 활동과 챌린지를 수행합니다.
2. 활동 결과에 따라 포인트가 적립됩니다.
3. 회원이 `해빛 마켓`에서 원하는 쿠폰 상품을 선택합니다.
4. 서버가 포인트 차감 가능 여부를 확인한 뒤 Giftishow API로 쿠폰 발급을 요청합니다.
5. 발급된 쿠폰은 앱의 `쿠폰 보관함`에 저장되며, 사용자는 바코드 또는 PIN으로 확인합니다.
6. 운영자는 `관제탑`에서 발급 상태를 확인하고 예외 건을 후속 처리합니다.

## 초기 운영 상품

- 메가MGC커피 `(ICE)아메리카노 모바일쿠폰`, 유효기간 60일, 2,000P
- 빽다방 `아메리카노(ICED) 모바일쿠폰`, 유효기간 60일, 2,000P

## 운영 메모

- 현재 쿠폰 교환은 온체인 자산이 아니라 포인트 기반으로 처리합니다.
- HBT는 자산 탭에서 별도 시스템으로 유지하며, 현재 쿠폰 교환 흐름과는 분리합니다.
- 쿠폰은 문자 기본 발송보다 앱 내 보관함 노출을 우선으로 설계했습니다.
- 상품 이미지는 Giftishow 원본 이미지를 사용하고, 브랜드 로고는 운영자가 승인한 실제 자산을 self-host 방식으로 반영합니다.
- 사용자 연락처는 Giftishow 발급 요건과 예외 대응을 위한 최소 범위에서만 사용합니다.

## 첨부 화면 구성

- 해빛 마켓 상품 목록 화면
- 쿠폰 보관함 화면
- 관제탑 보상 마켓 운영 화면

위 화면을 통해 사용자 교환 흐름과 관리자 운영 흐름을 함께 확인할 수 있습니다.
