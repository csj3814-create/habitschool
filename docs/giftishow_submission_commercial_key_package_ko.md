# 해빛스쿨 Giftishow 상용 Key 승인 제출 문서

## 서비스 개요

- 서비스명: 해빛스쿨(Habit School)
- 운영사: 공감케어
- 서비스 성격: 건강 습관 형성 보상형 앱
- 쿠폰 교환 자산: 앱 내 포인트
- 현재 운영 정책: 포인트 차감 후 Giftishow API 연동 쿠폰 발급
- 쿠폰 수령 방식: 앱 내 쿠폰 보관함 우선 노출
- 관리자 운영 방식: 관제탑에서 발급 상태, 예외 건, 수동 재확인 관리

## 서비스 이용 흐름

- 회원이 건강 챌린지와 앱 활동을 통해 포인트를 적립합니다.
- 회원이 해빛 마켓에서 원하는 리워드 상품을 선택합니다.
- 앱 서버가 포인트 차감 후 Giftishow API로 쿠폰 발급을 요청합니다.
- 발급된 쿠폰은 앱 내 쿠폰 보관함에서 바코드 이미지 또는 PIN으로 확인합니다.
- 운영자는 관제탑에서 발급 상태와 예외 건을 확인하고 후속 조치를 수행합니다.

## 초기 운영 상품

- 메가MGC커피 `(ICE)아메리카노 모바일쿠폰`, 60일, 2,000P
- 빽다방 `아메리카노(ICED) 모바일쿠폰`, 60일, 2,000P

## 운영 메모

- 포인트 기반 보상몰 운영 구조이며, 현재 기프티콘 교환은 HBT가 아닌 앱 내 포인트로 처리합니다.
- HBT는 자산 탭의 별도 시스템으로 유지하고, 현재 쿠폰 교환 흐름과는 분리합니다.
- 쿠폰은 기본적으로 앱 내 보관함에서 확인하도록 설계했고, 운영 예외 시에만 별도 재전송 정책을 검토합니다.
- 상품 이미지는 Giftishow 원본 상품 이미지를 사용하고, 브랜드 로고는 운영자가 승인한 실제 로고 자산을 self-host 방식으로 반영합니다.
- 사용자 연락처는 API 발급 요건 충족과 고객 지원을 위한 최소 범위에서만 사용합니다.
