# 해빛스쿨 보상 마켓 서비스 소개서

## 1. 서비스 개요

해빛스쿨(Habit School)은 식단, 운동, 수면, 마음건강 같은 일상 건강 활동을 기록하고
지속적인 건강 습관 형성을 돕는 모바일 기반 서비스입니다.

사용자는 앱 안에서 건강 활동을 수행하고 보상 포인트를 적립하며,
적립한 포인트를 `해빛 마켓`에서 모바일 쿠폰으로 교환할 수 있습니다.

현재 쿠폰 교환은 `포인트 차감` 기반으로 운영하며,
`HBT`는 자산 탭에서 별도로 유지되는 보조 자산 시스템으로 분리해 운영합니다.

## 2. Giftishow API 연동 목적

Giftishow Biz API는 해빛스쿨 보상 마켓의 실쿠폰 발급을 위해 사용합니다.

- 회원이 앱에서 원하는 리워드 상품을 선택합니다.
- 해빛스쿨 서버가 포인트 차감 조건을 확인한 뒤 Giftishow API로 쿠폰 발급을 요청합니다.
- 발급된 쿠폰은 문자 기본 발송보다 `앱 내 쿠폰 보관함` 노출을 우선합니다.
- 운영자는 관제탑에서 발급 상태와 예외 건을 확인하고 후속 조치를 진행합니다.

## 3. 현재 운영 정책

- 쿠폰 교환 자산: 포인트
- 교환 기준: 내부 교환 기준가
- 쿠폰 전달 방식: 앱 보관함 우선 노출
- 연락처 사용 범위: Giftishow 발급 요건 충족과 예외 처리 대응을 위한 최소 범위
- HBT 운영 방식: 자산 탭 별도 유지, 현재 쿠폰 교환 흐름과 분리

## 4. 주요 이용 흐름

1. 회원이 해빛스쿨 앱에서 건강 활동을 수행합니다.
2. 활동 결과에 따라 포인트가 적립됩니다.
3. 회원이 `해빛 마켓`에서 원하는 쿠폰 상품을 선택합니다.
4. 서버가 포인트 차감 가능 여부를 확인한 뒤 Giftishow API로 발급을 요청합니다.
5. 발급된 쿠폰은 앱의 `쿠폰 보관함`에 저장되고, 사용자는 바코드 또는 PIN으로 확인합니다.

## 5. 초기 운영 상품

- 메가MGC커피 `(ICE)아메리카노 모바일쿠폰`, 유효기간 60일, 2,000P 교환
- 빽다방 `아메리카노(ICED) 모바일쿠폰`, 유효기간 60일, 2,000P 교환

초기에는 소액 음료 쿠폰 중심으로 보상 흐름을 검증하고,
회원이 앱 안에서 적립과 교환 경험을 자연스럽게 이어갈 수 있도록 운영합니다.

## 6. 화면 및 자산 정책

- 상품 이미지는 Giftishow Biz 상품 원본 이미지를 사용합니다.
- 브랜드 로고는 운영자가 승인한 실제 로고 자산을 앱에서 self-host 방식으로 사용합니다.
- 해빛 마켓과 쿠폰 보관함은 모바일 우선 UI로 구성되어 있으며,
  상품 식별, 바코드 확인, PIN 확인, 유효기간 확인 흐름을 단순하게 유지합니다.

## 7. 운영 및 예외 처리

- 쿠폰 발급은 사용자 요청 시점에만 수행합니다.
- 발급 상태는 `pending_issue`, `issued`, `failed_manual_review`, `cancelled`로 관리합니다.
- 응답 누락이나 예외 상황이 발생하면 관제탑에서 `쿠폰 재조회`로 후속 확인을 진행합니다.
- 발급 이력, 포인트 차감, 예외 처리 이력은 서버 기준으로 관리합니다.

## 8. 기대 효과

- 건강 활동 보상이 실제 생활 혜택으로 이어집니다.
- 적립과 교환이 모두 앱 안에서 연결되어 사용자 경험이 단순합니다.
- 쿠폰 확인과 운영 관제를 분리해 사용자 경험과 운영 안정성을 함께 확보합니다.
