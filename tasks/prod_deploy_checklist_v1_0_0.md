# 해빛스쿨 v1.0.0 본서버 배포 체크리스트

## 1. staging 최종 확인
- [ ] 식단/운동/마음 저장이 실제로 빠르게 끝나는지 확인
- [ ] 저장 후 사진/영상/수면 캡처/공유 카드가 정상 복원되는지 확인
- [ ] 갤러리 반응 버튼과 댓글 입력이 모바일에서 잘리지 않는지 확인
- [ ] 친구 요청, 친구 수락, 친구 챌린지 시작 흐름 확인
- [ ] MetaMask / Trust Wallet 연결과 기존 앱 지갑 내보내기 흐름 확인
- [ ] 관리자 화면 핵심 수치와 회원 목록 확인

## 2. 코드 상태 확인
- [ ] `git status`가 배포 대상만 포함하는지 확인
- [ ] 로컬 캐시, 로컬 설정, 임시 메모가 커밋에 섞이지 않는지 확인
- [ ] `changelog.html`, 릴리즈 노트, 체크리스트가 최신 상태인지 확인

## 3. 배포 직전 검증
- [ ] `npm test`
- [ ] 필요한 번들/구문 체크
- [ ] staging 핵심 화면 수동 확인

## 4. 배포 순서
1. `git add`
2. `git commit`
3. `git push origin main`
4. 사용자에게 본서버 배포 확인 요청
5. 확인 후 `firebase deploy --only hosting,functions`

## 5. 배포 직후 확인
- [ ] 본서버 첫 접속 확인
- [ ] 기록 저장 / 공유 카드 / 갤러리 반응 확인
- [ ] 친구 요청 / 친구 챌린지 확인
- [ ] 자산 탭 / 외부 지갑 안내 확인
- [ ] 관리자 화면 / 주요 함수 로그 확인
