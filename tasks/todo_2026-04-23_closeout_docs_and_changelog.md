## Summary

### 식단 방법 코칭
- 프로필에서 식단 방법을 고르면 `내 기록` 가이드, 대시보드 요약, 식전/공복 알림이 선택한 방식에 맞게 바뀌도록 정리했다.
- 카드 문구와 배지를 모바일 기준으로 짧게 다듬고, 선택 저장 흐름과 알림 동의 모달도 단순하게 정리했다.
- `고단백 식단`은 최종적으로 제거했고, 기존 legacy 선택은 `저탄수 고단백 식단`으로 자연스럽게 이어지도록 맞췄다.

### 로그인 / PWA / 알림 딥링크 안정화
- Galaxy / Samsung Internet에서 반복 제보되던 Google 로그인 루프를 다시 정리해, 일반 브라우저 탭에서는 popup 경로를 우선 사용하도록 바꿨다.
- `홈 화면에 추가`는 native install prompt를 먼저 기다리도록 되돌려 원클릭 설치 체감을 복구했다.
- 푸시 알림의 `지금 기록` 버튼은 앱 창 우선 포커싱과 식단 슬롯 딥링크 처리까지 맞춰 실제 기록 화면으로 이어지게 보완했다.
- 서비스워커 등록 경로를 `/sw.js` 기준으로 정리해 콘솔의 불안정한 404 잡음을 줄였다.

### 갤러리 / 자산 / 관리자 대시보드
- 갤러리 새로고침 시 게스트 캐시가 남아 `모두의 최근 인증`이 비거나 내 것만 보이던 회귀를 고쳤다.
- 자산 탭 주황 요약 카드가 깨지던 CSS 파싱 회귀를 복구했다.
- 관리자 관제탑에는 `7일 일별 지급 포인트` 차트를 추가해 최근 운영 지표를 더 쉽게 볼 수 있게 했다.

### 명상 / 감사일기 경험 개선
- 마음 탭의 단순 체크박스를 가이드형 명상 카드 + 타이머 완료 흐름으로 바꿨다.
- `복식호흡`, `4-7-8 호흡`, `박스호흡`, `짧은 마음챙김` 4가지 방법을 지원하고, 호흡법 3종은 단계 표시와 단계별 비프 안내를 제공한다.
- 마음챙김은 영상 선택, 자동 전체화면, 랜덤 시작 지점, CSP 보강까지 연결해 같은 영상도 다른 구간을 즐길 수 있게 했다.
- 감사일기 입력부는 `명상하며 느낀 3줄 감사 일기`로 이름을 바꾸고, 지원 브라우저에서는 음성 입력으로 받아쓸 수 있게 했다.

### 운영 / 데이터 안정화
- 신규 가입 축하 `200P`가 누락되던 흐름을 찾아 다음 로그인에서 자동 복구되도록 정리했다.
- Firestore rules drift 때문에 본서버에서만 식단 방법 저장이 막히던 문제를 rules 배포 포함으로 바로잡았다.
- YouTube 이미지, Chart.js sourcemap, 서비스워커, CSS 분리, 대형 HTML/JS 변경 등 배포성 회귀를 잡기 위한 테스트와 lessons를 계속 보강했다.
