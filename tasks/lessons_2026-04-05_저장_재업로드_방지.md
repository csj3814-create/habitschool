# 2026-04-05 교훈 - 저장 시 재업로드 방지

## 규칙
- 파일 선택 즉시 업로드를 쓰는 화면에서는 저장 후 반드시 input 값을 비워야 한다.
- 저장 로직은 `input.files[0]`만 보지 말고, 이미 준비된 pending 업로드 결과와 저장된 URL을 먼저 확인해야 한다.
- 사진만 pre-upload하고 영상은 저장 시점에 올리면 저장 체감이 급격히 느려진다. 영상도 같은 규칙으로 맞춘다.
- 날짜별 로그를 저장할 때는 방금 로드한 문서를 메모리 캐시로 재사용하고, 불필요한 재조회로 저장 버튼을 붙잡지 않는다.
