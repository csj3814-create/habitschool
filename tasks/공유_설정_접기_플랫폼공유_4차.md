# 2026-04-04 공유 설정 접기 · 플랫폼 공유 4차
> **상태**: 완료

## 목표
- 기본 공유 안내 박스의 모바일 넘침과 과한 둥근 외곽선을 정리한다.
- 공개 설정을 `공유 설정 열기` 버튼 아래로 옮기고, 눌렀을 때만 펼쳐지게 만든다.
- 숨길 항목에 `식단`, `운동`, `마음`을 포함해 실제 공유 범위 제어를 넓힌다.
- 인스타그램, 페이스북, X, 블로그, 카카오톡 공유 흐름을 더 쉬운 방식으로 정리한다.

## 반영 내용
- 안내 박스를 버튼과 분리해 `안내 > 공유 설정 열기 > 공개 설정 패널` 순서로 재배치했다.
- 공개 설정 패널은 기본 접힘 상태로 두고, 버튼을 누르면 열리고 다시 누르면 닫히게 만들었다.
- 숨김 옵션을 `이름 / 날짜 / 식단 / 운동 / 포인트 / 마음`으로 확장했다.
- 공유 카드와 갤러리 카드 모두 식단, 운동, 마음 항목 숨김이 실제 반영되도록 맞췄다.
- 공유 모달에 `블로그` 버튼과 `캡션 복사` 버튼을 추가했다.
- 플랫폼 공유는 `모바일: 시스템 공유 우선`, `PC: 이미지 저장 + 캡션 복사 + 플랫폼 창 열기` 방식으로 정리했다.
