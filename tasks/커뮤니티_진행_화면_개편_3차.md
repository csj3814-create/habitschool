# 2026-04-04 커뮤니티/진행 화면 개편 3차
> 상태: 완료

## 작업
- 주간 미션 진행 영역을 요약 중심으로 정리하고 상세는 접어서 보도록 조정
- 친구 활동, 소셜 챌린지 목록을 처음에는 짧게만 보여주고 더 보기 버튼으로 확장
- 커뮤니티 카드와 모바일 여백, 버튼 크기를 더 촘촘하게 다듬기

## 결과
- 미션 진행 카드는 `달성률 / 남은 시간 / 이번 주 미션`만 먼저 보이고, 상세 막대는 펼쳤을 때만 보이도록 정리했다.
- 친구 활동은 활동량이 높은 순으로 정렬하고 3개만 먼저 노출하도록 바꿨다.
- 소셜 챌린지는 초대 대기 항목을 먼저 보여주고 2개만 먼저 노출하도록 바꿨다.
- 월간 커뮤니티 카드는 설명과 CTA를 더 짧게 줄여 바로 행동이 보이게 정리했다.
