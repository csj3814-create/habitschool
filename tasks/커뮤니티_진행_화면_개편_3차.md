# 2026-04-04 커뮤니티/진행 화면 개편 3차
> 상태: 완료

## 작업
- 주간 미션 진행 영역을 기본 요약 + 접힘 상세 구조로 재배치
- 친구 활동/소셜 챌린지 리스트를 기본 2~3개만 보이도록 축소
- 월간 커뮤니티 카드의 상세 정보와 지난 흐름 링크를 접힘 영역으로 이동
- 모바일에서 커뮤니티 액션 카드와 모듈 카드의 세로 길이를 줄이도록 여백 조정

## 결과
- 주간 미션 카드는 기본 상태에서 달성률, 남은 일수, 위험 상태만 빠르게 읽히고, 세부 미션은 펼쳤을 때만 보이게 바뀌었다.
- 친구 활동과 챌린지는 기본 노출 수를 줄이고 `더 보기`로 이어지게 해서 커뮤니티 영역 전체 높이를 줄였다.
- 월간 커뮤니티 카드는 핵심 지표만 먼저 보여주고, MVP/카테고리 킹/지난 흐름은 상세 보기 안으로 들어갔다.
- 모바일에서는 커뮤니티 액션 카드가 2열 유지, 카드 패딩 축소, 버튼 높이 축소로 첫 화면 압박을 줄였다.
