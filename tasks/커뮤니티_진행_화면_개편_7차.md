# 커뮤니티/진행 화면 개편 7차

## 목표
- 갤러리 사용자 필터 바를 더 짧고 명확한 상태 표시로 정리
- 갤러리 카드 상단 사용자 정보 영역을 더 압축해서 빠르게 스캔되게 만들기

## 반영 내용
- 사용자 필터 바를 `개인 필터` 상태 칩, 사용자 이름, `전체 보기` 버튼 구조로 재정리
- 필터 문구를 긴 설명형 문장 대신 사용자 이름 중심으로 축약
- 갤러리 카드 상단 이름 줄에 `친구` 칩과 스트릭 배지를 같이 묶고, 날짜는 별도 짧은 줄로 분리
- 아바타, 친구 버튼, 메뉴 버튼 크기와 간격을 더 줄여 카드 헤더 높이를 압축
- 다크모드와 모바일에서도 같은 구조가 읽히도록 필터 바/칩 스타일을 같이 맞춤

## 기대 효과
- 지금 어떤 개인 필터가 걸려 있는지 더 빨리 이해할 수 있음
- 카드 상단에서 작성자, 친구 여부, 스트릭을 한 줄에서 더 빠르게 읽을 수 있음
- 모바일에서 카드 헤더가 덜 길어져 피드 스캔 속도가 좋아짐

## 검증
- `npm test`
- `node --check`로 `js/app.js` 문법 확인
- localhost 응답 및 핵심 클래스 반영 확인
- staging 배포 후 홈 응답 확인

## 다음 작업
- 갤러리 상단 필터 칩 우선순위를 실제 사용 흐름 기준으로 재배치할지 검토
- 카드 상단 친구 버튼을 더 명확한 상태 라벨로 바꿀지 점검
