# 2026-04-04 기본 공유 헤더 정리 3차
> **상태**: 완료

## 목표
- 갤러리 카드 첫 줄 길이를 줄여 이름과 상태가 더 빨리 읽히게 만든다.
- 친구 버튼과 메뉴 버튼은 보이되, 기록 정보보다 덜 튀게 정리한다.

## 반영 내용
- 카드 헤더를 `이름 한 줄 + 상태 줄 + 우측 액션 묶음` 구조로 재정렬했다.
- 날짜, 친구 표시, 스트릭을 작은 상태 줄로 모아 첫 줄을 짧게 만들었다.
- 친구 버튼과 메뉴 버튼 크기를 줄여 헤더 시선을 덜 빼앗게 조정했다.
- 다크모드에서도 날짜 칩과 관계 칩이 과하게 밝지 않도록 정리했다.

## 검증
- `npm test` 통과
- `node --check` 대체 `.mjs` 문법 확인 통과

## 다음 후보
- 갤러리 카드 본문 첫 줄의 포인트/유형 칩 우선순위 다시 정리하기
- 공유 카드 설정 패널을 더 짧은 칩형 토글로 바꾸기
- 친구 추가 버튼을 상태형 아이콘 버튼으로 더 압축할지 검토하기
