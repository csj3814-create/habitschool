# 2026-04-04 공유 기본 + 정보 가리기 개편 1차
> **상태**: 완료

## 목표
- 해빛 기록은 기본 공유로 두고, 원치 않는 정보만 사용자가 직접 가릴 수 있게 바꾼다.
- 공유 정책이 UI 문구에만 머무르지 않고 저장 데이터와 갤러리 표시에도 실제 반영되게 한다.

## 반영 내용
- 커뮤니티/갤러리 상단 공개 안내를 `기본 공유` 기준으로 변경했다.
- 공유 카드에 공개 설정 패널을 추가했다.
- 가릴 수 있는 항목은 `이름`, `날짜`, `포인트`, `마음 글` 4가지다.
- 공유 설정은 `daily_logs.shareSettings`로 저장되며 날짜 이동 시 다시 불러온다.
- 갤러리 카드도 같은 설정을 읽어 익명 표시, 날짜 숨김, 포인트 배지 숨김, 마음 글 비노출을 반영한다.

## 구현 포인트
- `index.html`
  - 공개 안내 문구를 `기본 공유` 기준으로 수정
  - 공유 카드 안에 공개 설정 패널 추가
- `js/app.js`
  - 공유 설정 기본값/정규화 헬퍼 추가
  - 공유 카드 렌더링이 `shareSettings`를 반영하도록 수정
  - 저장 시 `daily_logs.shareSettings` 포함
  - 불러오기/초기화 시 공개 설정 복원 및 기본값 재설정 연결
  - 갤러리 카드가 숨김 설정을 읽어 표시 형태를 바꾸도록 수정
- `styles.css`
  - 공개 설정 패널, 숨김 상태, 모바일 레이아웃 스타일 추가
- `firestore.rules`
  - `daily_logs.shareSettings` 허용 및 bool 검증 추가
