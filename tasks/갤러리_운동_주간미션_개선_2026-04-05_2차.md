# 2026-04-05 갤러리 · 운동 · 주간 미션 개선 2차

## 작업 목표
- 갤러리 공유 영역의 시각 밀도를 줄이고 공유 카드 사진 영역을 키운다.
- weekly spotlight, 최근 인증, 댓글, 하단 CTA를 모바일과 큰 화면 모두 더 간결하게 정리한다.
- 운동 저장 흐름에서 8천보 포인트, 운동 영상 첫 박스 재사용을 정상화한다.
- 주간 미션 승급 직후 즉시 재설정되지 않고 다음 날 다시 고를 수 있게 만든다.

## 반영 내용
- 공유 템플릿 라벨을 제거하고 템플릿 버튼을 더 얇은 pill 형태로 축소했다.
- 공유 카드 상단 칩, 제목, 부제, 태그를 압축하고 사진 영역을 더 크게 잡았다.
- weekly spotlight 헤더가 큰 화면에서도 한 줄을 유지하도록 조정했다.
- 최근 인증 설명 문구를 제거하고 필터 칩 간격을 줄여 잘리지 않게 정리했다.
- 댓글은 최근 5개까지 먼저 보여주고, 이후에는 `댓글 모두 보기`로 펼치게 바꿨다.
- 갤러리 하단 CTA를 `해빛스쿨 단톡방 참여하기`로 되돌리고 노란색 중앙 정렬로 맞췄다.
- 공유 준비 안내 문구와 기록/가이드 문구를 모바일 기준 한 줄에 가깝게 줄였다.
- 운동 포인트는 걸음수 8,000보 이상일 때만 반영되게 수정했다.
- 운동 영상 업로드 시 비어 있는 첫 번째 운동 영상 박스를 우선 재사용하게 했다.
- 레벨 승급 성공 시 `missionLevelUpDate`를 기록하고, 같은 날에는 새 주간 미션을 다시 고르지 못하게 막았다.
- 승급 성공 후에는 alert 대신 토스트와 대시보드 재렌더를 사용해 실제 상태가 바로 보이게 했다.
- 운동 영상은 로컬에서 추출한 썸네일까지 저장 시 함께 업로드해, 공유 카드에서 파란 플레이스홀더 대신 실제 썸네일이 보이게 보강했다.
- 프로필 탭의 업데이트 노트는 본문 노출을 빼고 `changelog.html`로 같은 창에서 보게 되돌렸다.

- 식단·운동·마음 탭은 `가이드 → 포인트 바 → 본문` 순서를 유지해 세 탭의 첫 화면 패턴을 같게 맞춘다.
