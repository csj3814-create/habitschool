# 2026-04-06 전체폭 버튼 sizing 교훈

## 패턴

- 카드 안에서 `width: 100%` 버튼을 만들 때 `padding`이 있으면 기본 `content-box` 때문에 실제 렌더 폭이 부모보다 커질 수 있다.
- 모바일에서는 이 문제가 바로 오른쪽 튀어나옴으로 보인다.

## 규칙

- 카드 안 전체폭 버튼은 기본으로 아래 규칙을 같이 넣는다.
  - `box-sizing: border-box`
  - `max-width: 100%`
  - `min-width: 0`
- 긴 CTA를 새로 만들 때는 `width: 100%`만 넣고 끝내지 않는다.
