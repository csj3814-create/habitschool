# 커뮤니티/진행 화면 개편 6차

## 목표
- 갤러리 카드 한 장의 세로 길이를 줄여 더 많이 한눈에 보이게 만들기
- 반응 버튼과 댓글 미리보기 영역을 더 빠르게 읽히게 정리

## 반영 내용
- 반응 버튼을 4칸 정렬 구조로 바꿔 한 줄에서 안정적으로 읽히게 정리
- 버튼 안 숫자를 작은 카운트 배지로 분리해 반응 수를 더 빨리 스캔할 수 있게 조정
- 중복이던 반응 요약 줄은 제거
- 댓글 미리보기는 기본 1개만 보이고 나머지는 `모두 보기`로 확장되게 변경
- 댓글 입력 영역과 카드 헤더/메타 여백을 소폭 줄여 카드 전체 높이를 압축

## 기대 효과
- 같은 화면 높이에서 더 많은 인증 카드를 볼 수 있음
- 반응과 댓글 상태를 버튼 줄만 보고 빠르게 이해할 수 있음
- 댓글이 많은 카드도 기본 높이가 과하게 커지지 않음

## 검증
- `npm test`
- `node --check`로 `js/app.js` 문법 확인
- localhost 응답 및 핵심 문구 확인
- staging 배포 후 홈 응답 확인

- 반응 버튼 컬러 대비와 다크모드 카운트 배지 가독성 추가 점검
