# 2026-04-06 외부 지갑 전환 결과

## 목표
- 자동 생성 내장 지갑 중심 구조를 줄이고, 사용자가 직접 연결하는 외부 지갑 중심 구조로 전환
- `MetaMask / Trust Wallet` 연결을 우선 지원
- 기존 앱 지갑 사용자를 위한 이전 경로를 따로 마련

## 반영한 내용

### 1. 사용자 문서의 지갑 필드 확장
- `externalWalletAddress`
- `walletConnectionMode`
- `walletProviderType`
- `walletChainId`
- `walletConnectedAt`

### 2. 자산 탭 지갑 UX 정리
- MetaMask 연결
- Trust Wallet 연결
- 지갑 연결 해제
- 현재 연결 주소 표시
- 연결 방식에 따른 상태 문구 분리

### 3. 기존 앱 지갑 사용자 이전 경로
- `기존 앱 지갑 내보내기` 버튼 추가
- 개인키 1회 보기/복사 제공
- MetaMask / Trust Wallet 가져오기 안내 연결

### 4. 온체인 로직 정리
- 온체인 조회와 챌린지 예치 흐름에서 `externalWalletAddress` 우선 사용
- functions / rules / 프런트 상태를 같은 기준으로 정리
