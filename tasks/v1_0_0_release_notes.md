# 해빛스쿨 v1.0.0 릴리즈 노트

## 릴리즈 정의
- 버전명: `v1.0.0`
- 기준: staging에서 장기간 다듬은 UI/기록/친구/공유/지갑 흐름을 본서버 배포 후보로 정리한 첫 정식 버전
- 메인넷 전환: 이번 버전 범위에서 제외

## 이번 버전의 핵심 변화

### 1. 기록 경험 재구성
- 내 기록 탭 흐름을 `오늘의 루틴 -> 오늘의 인증 현황 -> 이번 주 흐름 -> 나의 주간 미션` 중심으로 재정리
- 식단/운동/마음 탭의 저장 흐름과 가이드를 모바일 기준으로 단순화
- 사진, 영상, 텍스트 재저장 시 불필요한 재업로드를 줄여 저장 속도를 개선

### 2. 갤러리와 공유 카드 개편
- 갤러리 피드와 공유 카드 구조를 모바일 기준으로 압축
- 반응, 댓글, 필터, 공유 템플릿, 공개 범위 안내를 사용자 행동 중심으로 다시 구성
- 공유 카드 이미지와 동영상 썸네일 처리 안정화

### 3. 친구 연결과 친구 챌린지 복구
- 초대 링크, 초대 코드, 친구 요청/수락 흐름 정리
- 친구 챌린지 카드 재도입
- 친구별 오늘 인증, 이번 주 인증, 최근 30일 기록 기준으로 챌린지 가능 여부 표시

### 4. 자산 탭과 블록체인 흐름 정리
- 외부 지갑 연결 기준으로 `MetaMask / Trust Wallet` 흐름 정리
- 기존 앱 지갑 사용자를 위한 `개인키 1회 보기/복사` 내보내기 기능 추가
- HBT 잔액, 포인트 기록, 챌린지 예치 UX 보강

### 5. 운영/배포 체계 안정화
- staging Firebase 프로젝트와 emulator 기반 사전 검증 체계 정착
- changelog, 프로필 업데이트 노트, 릴리즈 체크리스트 정리
- 로컬 캐시/로컬 설정 같은 배포 비대상 파일 정리 준비

## 이번 버전에서 의도적으로 제외한 것
- BNB Chain 메인넷 이전
- WalletConnect 실연결
- 메인넷 토큰 주소 안내와 운영 전환

## 배포 전 최종 확인 포인트
- 저장/공유/친구 챌린지/외부 지갑 연결 핵심 흐름 재확인
- 본서버 배포 후 초기 안정화 기간 동안 온체인 전환은 보류
- 메타마스크/트러스트월렛 연결과 기존 앱 지갑 내보내기 안내 문구 재확인
