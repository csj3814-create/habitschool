# 관제탑 회원 목록 상세 열 위치 변경

## 목표

- 회원 목록의 `상세` 열을 `온체인 HBT` 다음, `공복혈당` 앞으로 이동한다.
- 상세 버튼의 동작과 회원 데이터 조회·정렬 로직은 변경하지 않는다.

## 현재와 변경 후 순서

- 현재: `온체인 HBT → 공복혈당 → 혈압 → 메시지 → 미활동 이메일 → 상세`
- 변경: `온체인 HBT → 상세 → 공복혈당 → 혈압 → 메시지 → 미활동 이메일`

## 예상 수정 파일

- `admin.html`: 표 헤더와 각 회원 행의 상세 셀을 같은 위치로 이동한다.
- `tests/report-admin-detail-ui.test.js`: 헤더와 행의 열 순서가 일치하는지 회귀 테스트를 추가한다.
