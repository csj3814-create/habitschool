# 2026-04-07 Floating Bar Visual Centering

- A floating CTA can be mathematically centered and still look wrong if its width is too narrow for the visible content column.
- When users say a bottom CTA or install banner is left-shifted, check both `left` and `usable width` together.
- For long fixed CTAs, prefer a wide viewport-centered bar and reset stale inline positioning before reporting the fix.
- Do not claim this class of UI issue is fixed without refreshing real browser assets and visually checking the page.
