# 2026-04-06 친구 챌린지 모듈 스코프 교훈

- `type="module"` 스크립트에서는 `foo = function(){}` 같은 대입식 재정의를 쓰지 않는다.
- 같은 이름 함수를 뒤에서 다시 바꿔야 하면 `function foo(){}` 선언으로 덮거나, 이름을 분리해 호출 지점을 명시적으로 교체한다.
- 전역 카드 렌더 함수가 비어 보이면 UI 로직만 보지 말고 브라우저 콘솔의 `ReferenceError`부터 확인한다.
- staging 배포 전에는 `esbuild` 번들 체크만 하지 말고, 브라우저 콘솔 기준으로 함수 이름 누락이 없는지 한 번 더 확인한다.
