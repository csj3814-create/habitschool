업데이트 노트

해빛스쿨의 주요 변경과 개선 기록
최신 업데이트: 2026년 4월 20일
v1.0.7 2026년 4월 20일
웹과 PWA 중심 흐름을 다듬고 업로드 체감 속도와 챌린지 정산 정확도를 보완했습니다
  • 개선 앱 설치 흐름을 웹과 홈 화면 설치 중심으로 정리해, 브라우저에서 바로 이해하고 설치할 수 있게 했습니다.
  • 개선 작은 일반 사진은 불필요한 전처리 압축을 줄여 더 빠르게 업로드되도록 다듬었습니다.
  • 개선 3~4MB 정도의 작은 운동 영상은 원본 업로드가 먼저 붙도록 조정해 저장 대기 체감을 줄였습니다.
  • 수정 7일 챌린지 마지막 날 기록과 완료일 계산이 어긋나며 실패로 보이던 정산 흐름을 바로잡았습니다.
  • 수정 서비스워커와 버전 자산을 다시 정리해 최신 웹 코드가 더 안정적으로 반영되게 했습니다.
v1.0.6 2026년 4월 12일
메인넷 운영 안정화와 업로드·연결 흐름을 한 번 더 다듬었습니다
  • 개선 내 자산 탭에서 HBT 거래 기록을 메인넷 기준으로 더 안정적으로 불러오고, 느린 온체인 확인 중에도 카드가 오래 비어 있지 않게 다듬었습니다.
  • 개선 포인트와 HBT 잔액이 일시적으로 늦게 들어와도 마지막 정상값을 먼저 보여주고, 백그라운드 재시도로 더 빨리 복구되게 했습니다.
  • 개선 사진·영상 업로드는 저장 후 백그라운드로 이어지게 하고, 각 파일마다 업로드 진행률을 바로 확인할 수 있게 했습니다.
  • 수정 같은 날 새 사진이나 영상을 추가해도 기존 기록 미디어가 함께 사라지지 않도록 업로드 병합 방식을 고쳤습니다.
  • 개선 해빛코치 계정 연결 카드에서 카카오톡 1:1 채팅을 바로 열 수 있게 하고, 연결 후 최근 !연결 이력이 더 안정적으로 남도록 정리했습니다.
  • 수정 카카오 표시 이름을 받지 못한 경우에도 `표시 이름 미확인` 대신 현재 1:1 계정으로 자연스럽게 안내하도록 바꿨습니다.
v1.0.5 2026년 4월 12일
메인넷 전환 기준으로 내 지갑과 토크노믹스 안내를 다시 정리했습니다
  • 개선 내 지갑 탭에서 활성 체인 배지, HBT 컨트랙트 링크, 챌린지 컨트랙트 링크를 바로 확인할 수 있게 했습니다.
  • 개선 포인트 → HBT 변환 안내를 온체인 currentRate 기준으로 바꾸고, 초기 1:1처럼 보이던 기본 문구를 정리했습니다.
  • 개선 반감기 카드와 거래 기록 빈 상태 문구를 메인넷 운영 흐름에 맞게 다시 다듬었습니다.
  • 개선 HBT 토크노믹스 페이지에 메인넷 컨트랙트 주소와 메인넷 전환 시점 비율(1P = 4 HBT)을 반영했습니다.
  • 수정 외부 지갑 연결, 기존 앱 지갑 내보내기, 메인넷 보관 안내가 서로 다른 표현으로 보이던 부분을 하나로 맞췄습니다.
v1.0.4 2026년 4월 10일
챌린지 기준을 쉽게 정리하고 앱 사용 흐름을 다듬었습니다
  • 개선 위클리와 마스터 챌린지 안내를 더 쉽게 바꿔서, 하루 65점 기준과 보상 조건을 한눈에 볼 수 있게 했습니다.
  • 개선 내 지갑과 토크노믹스 화면의 한도와 안내 문구를 실제 운영 기준에 맞게 다시 정리했습니다.
  • 수정 구글 로그인 뒤에 첫 화면에서 잠깐 멈추던 흐름을 줄여서 앱으로 더 자연스럽게 들어오게 했습니다.
  • 개선 운동 동영상을 고른 직후 바로 저장 중으로 보이지 않게 하고, 저장 버튼을 눌렀을 때만 진행 상황이 보이게 했습니다.
  • 개선 외부 지갑 대신 앱 지갑을 기본 사용 방식으로 다시 정리해 더 쉽게 쓸 수 있게 했습니다.
  • 개선 메인넷 준비용 서버 점검과 배포 전 확인 항목을 다시 정리했습니다.
v1.0.3 2026년 4월 9일
간편 모드, 마일스톤 정리, Android 공유 보완
  • 새 기능 `/simple` 간편 모드를 추가하고 식단, 운동, 마음만 보이는 진입 흐름을 만들었습니다.
  • 개선 간편 프로필에 현재 포인트, 친구 초대 QR, 해빛스쿨 단톡방 QR, 앱 설치 버튼을 정리했습니다.
  • 개선 간편 모드 문구와 상단 구성을 더 단순하게 바꾸고 식단과 마음의 AI 분석 버튼을 다시 보이게 했습니다.
  • 수정 기존 회원에게 시작 마일스톤 보상이 다시 뜨던 문제를 정리했습니다.
  • 수정 Android 사진 공유가 자동 저장되지 않더라도 식단 화면으로 바로 들어가도록 보완했습니다.
v1.0.2 2026년 4월 8일
Android 연동, 공유 안내, TWA 준비
  • 새 기능 Android에서 Health Connect 걸음수 연동과 관련 안내 흐름을 보강했습니다.
  • 수정 Android 공유 목록 노출과 앱 아이콘 여백을 조정해 삼성 기기 표시 문제를 보완했습니다.
  • 개선 공유한 사진이 식단 기록 흐름으로 자연스럽게 이어지도록 보완했습니다.
  • 개선 release signing, assetlinks, fullscreen TWA 준비 문서를 다음 단계로 이어지게 정리했습니다.
  • 수정 기존 회원 APK 설치 후 갤러리 반응과 포인트 계산 문제를 추가로 보완했습니다.
v1.0.1 2026년 4월 7일
설치 흐름, 알림, AI 분석 저장 보강
  • 개선 설치 배너와 하단 버튼에서 앱 설치를 더 분명하게 안내하고, iOS 홈 화면 설치 흐름도 보강했습니다.
  • 새 기능 프로필에서 직접 켜는 알림 권한 카드와 친구·챌린지 관련 알림 흐름을 추가했습니다.
  • 수정 식단 AI 분석 결과가 저장되고 새로고침 뒤에도 다시 보이도록 보완했습니다.
  • 개선 친구 챌린지 카드에서 바로 시작, 대기, 취소를 처리하기 쉽게 흐름을 정리했습니다.
  • 수정 친구 연결과 소셜 챌린지 권한 쪽 배포 문제를 보완했습니다.
v1.0.0 2026년 4월 6일
