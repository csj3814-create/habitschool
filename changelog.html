해빛스쿨의 주요 변경과 개선 기록
최신 업데이트: 2026년 4월 27일
v1.0.9 2026년 4월 27일
해빛 마켓 시각 자산과 포인트 교환 안내를 정리했습니다
  • 신규 해빛 마켓 상품 카드와 쿠폰 보관함에 기프티쇼 원본 상품 이미지를 연결했습니다.
  • 수정 메가MGC커피와 빽다방 로고를 승인된 실제 PNG 자산으로 교체해 화면 일관성을 맞췄습니다.
  • 수정 보상 마켓 일일 교환 한도 계산을 바로잡아 2,000P 상품이 잘못 막히던 문제를 해결했습니다.
  • 개선 해빛 마켓 안내 문구를 포인트 교환과 앱 보관함 수령 중심으로 더 간단하게 정리했습니다.
v1.0.8 2026년 4월 23일
식단 코칭, 명상 가이드, 로그인 안정화를 한 번에 반영했습니다
  • 신규 프로필에서 고른 식단 방식에 맞춰 기록 가이드, 대시보드 요약, 식전 알림 문구를 정리했습니다.
  • 개선 명상과 호흡 가이드를 더 자연스럽게 다듬고 단계 표현과 사운드 안내를 보완했습니다.
  • 신규 감사일기 박스를 명상 흐름과 맞는 구조로 정리하고 브라우저에서 접근하기 쉽게 개선했습니다.
  • 수정 Galaxy 계열 브라우저에서 반복되던 Google 로그인 루프를 줄이도록 로그인 경로를 다시 정리했습니다.
  • 수정 푸시 알림의 `지금 기록` 버튼이 실제 기록 화면으로 이어지도록 딥링크 연결을 보완했습니다.
v1.0.7 2026년 4월 20일
설치 안내, 업로드 체감, 챌린지 정산 정확도를 보완했습니다
  • 개선 앱 설치 흐름을 다시 정리해 브라우저에서 더 쉽게 이해하고 진행할 수 있게 했습니다.
  • 개선 사진과 영상 업로드의 전처리 부담을 줄여 더 빠르게 올릴 수 있게 했습니다.
  • 개선 배경 업로드 진행 표시를 다듬어 업로드 체감이 더 안정적으로 보이도록 했습니다.
  • 수정 7일 챌린지 마지막 날 기록 후 정산이 어긋나던 문제를 바로잡았습니다.
  • 수정 서비스워커 버전과 자산 캐시를 정리해 최신 코드 반영이 더 안정적으로 이뤄지게 했습니다.
v1.0.6 2026년 4월 12일
자산 화면과 업로드 흐름, 커뮤니티 연결을 손봤습니다
  • 개선 자산 탭의 HBT 기록과 요약을 더 안정적으로 불러오도록 정리했습니다.
  • 개선 로그인 직후에도 최근 요약 값이 먼저 보이고, 뒤에서 최신 값으로 자연스럽게 갱신되도록 했습니다.
  • 개선 사진과 영상 업로드 진행을 화면에서 바로 확인할 수 있도록 개선했습니다.
  • 수정 같은 날 여러 기록을 남겨도 기존 미디어가 잘못 사라지지 않도록 업로드 병합 방식을 보완했습니다.
  • 개선 해빛코치 계정 연결과 1:1 상담 진입을 더 쉽게 정리했습니다.

앞으로도 더 안정적이고 쓰기 쉬운 건강 습관 앱으로 계속 다듬겠습니다.
