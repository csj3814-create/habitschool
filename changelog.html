해빛스쿨의 주요 변경과 개선 기록
마지막 업데이트: 2026년 7월 17일
v1.0.18 2026년 7월 17일
쿠폰 발급과 다시 받기 기능을 안정화했습니다
  • 수정 쿠폰은 정상 발급됐지만 앱 보관함에 발급 대기로 남을 수 있던 문제를 고쳤습니다.
  • 신규 실발급 쿠폰을 저장된 연락처로 다시 받을 수 있는 문자로 다시 받기 기능을 추가했습니다.
  • 수정 발급 결과가 불확실한 경우 새 쿠폰을 중복 주문하지 않고 기존 발급 상태를 먼저 확인하도록 보강했습니다.
  • 개선 관제탑 보상 마켓에서 최근 발급 이력을 더 빠르게 확인하고 필요할 때 기존 쿠폰 MMS를 다시 보낼 수 있습니다.
  • 수정 앱 캐시 버전을 v240으로 갱신해 최신 보상 마켓 화면이 안정적으로 반영되도록 했습니다.
v1.0.17 2026년 6월 25일
영어 버전 진입점과 simple 앱 디자인을 개선했습니다
  • 신규 `/en`을 영어 공식 진입점으로 정리하고 Food, Exercise, Mind, Profile 중심의 simple 앱을 제공합니다.
  • 개선 영어 simple 화면의 카드, 업로드 영역, AI 버튼, CTA 색상을 한국어 simple 화면과 더 어울리도록 다듬었습니다.
  • 수정 영어 Exercise와 Mind 화면에 남아 있던 한국어 문구를 정리했습니다.
  • 수정 앱 캐시 버전을 v216으로 올려 최신 영어 화면이 안정적으로 반영되도록 했습니다.
v1.0.16 2026년 6월 4일
운동 소모임과 단톡방 진입 흐름을 다듬었습니다
  • 개선 대시보드에서 소모임을 최대 4개까지 볼 수 있도록 조정했습니다.
  • 개선 갤러리의 단톡방 참여 버튼은 계정 연결 없이 바로 오픈채팅방을 열도록 정리했습니다.
  • 수정 앱 캐시 버전을 v208로 갱신해 최신 화면이 안정적으로 반영되도록 했습니다.
v1.0.15 2026년 6월 4일
운동 소모임 시범 운영 기능을 추가했습니다
  • 신규 만보 걷기, 홈트 인증방, 헬스장 출석, 러닝 클럽 소모임을 추가했습니다.
  • 신규 사용자는 최대 2개 소모임에 참여할 수 있고, 각 소모임의 100일 보상 진행도는 따로 쌓입니다.
  • 신규 모임장이 제출 기록을 확인하고 승인 또는 반려할 수 있는 검토 화면을 추가했습니다.
  • 개선 소모임 참여 시 200P 예치, 100일 승인 완료 시 3,000P 보상 구조를 반영했습니다.
  • 수정 같은 날짜에 2개 소모임 조건을 모두 만족하면 각 소모임 진행도가 각각 1회씩 오르도록 고쳤습니다.
v1.0.14 2026년 6월 3일
동영상 업로드와 모임장 검토 미디어를 안정화했습니다
  • 수정 일부 삼성 인터넷에서 운동 영상 업로드가 1%에 머무는 문제를 줄였습니다.
  • 수정 모임장 검토 화면에서 사진은 크게 보고, 영상은 실제 영상 파일이 있을 때만 재생되도록 고쳤습니다.
  • 개선 영상 썸네일만 있는 오래된 제출 기록은 관련 일일 기록에서 원본 영상 주소를 보강해 보여줍니다.
v1.0.13 2026년 6월 1일
새 날짜 리로드와 기록 유지 흐름을 보강했습니다
  • 수정 오랜만에 앱을 열었을 때 새 날짜 기준으로 더 정확하게 다시 불러오도록 개선했습니다.
  • 수정 같은 날 사진을 추가로 올린 뒤 앞선 기록이 사라져 보일 수 있는 흐름을 점검하고 보강했습니다.
  • 개선 날짜가 바뀐 뒤에도 선택 날짜와 화면 상태가 어긋나지 않도록 갱신 조건을 정리했습니다.
v1.0.12 2026년 5월 26일
주간 챌린지 재시작과 PWA 갱신을 안정화했습니다
  • 수정 완료한 주간 챌린지를 같은 날 다시 시작할 때 이미 1일차로 잡히던 문제를 고쳤습니다.
  • 개선 보상 수령 뒤 자산 화면이 오래된 챌린지 상태를 보여주지 않도록 서버 기준 새로고침을 강화했습니다.
  • 수정 PWA 캐시 버전을 갱신해 최신 코드가 더 안정적으로 반영되도록 했습니다.
v1.0.11 2026년 5월 19일
쿠폰 보관함과 마음 힌트 흐름을 정리했습니다
  • 개선 사용 완료 또는 만료된 쿠폰은 보관함에서 사진과 바코드를 접고 핵심 정보만 보이도록 정리했습니다.
  • 개선 마음 기록의 힌트 안내와 명상 흐름을 더 짧고 자연스럽게 다듬었습니다.
  • 수정 오래된 쿠폰이 자동 정리되도록 서버 정리 작업을 보강했습니다.
v1.0.10 2026년 5월 1일
로딩 안정성과 관리자 화면을 개선했습니다
  • 수정 자산 탭에서 Firestore가 느릴 때 거래 기록과 HBT 정보가 무한 로딩처럼 보이던 문제를 줄였습니다.
  • 개선 친구, 커뮤니티, 갤러리 같은 보조 데이터 로딩이 화면 전체를 막지 않도록 정리했습니다.
  • 수정 관리자 보상마켓 운영에 필요한 Firestore 권한 배포를 정리했습니다.
v1.0.9 2026년 4월 29일
쿠폰 유효기간과 표시 정보를 정확하게 맞췄습니다
  • 수정 아메리카노 쿠폰 유효기간을 실제 발급 기준과 맞췄습니다.
  • 개선 쿠폰 보관함에서 만료 날짜와 남은 일수를 더 쉽게 확인할 수 있도록 정리했습니다.
  • 수정 쿠폰 상태 조회 응답에서 만료일을 우선 반영하도록 고쳤습니다.
