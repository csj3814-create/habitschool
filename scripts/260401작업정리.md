**원인**: 티어 카드 전체 `onclick`이 touch 이벤트에서 `stopPropagation` 타이밍 문제로 슬라이더 조작 시 `startChallenge30D` 호출

**수정**: `index.html` — onclick을 카드 전체 → `.tier-card-header`(아이콘+제목 영역)로 분리

### 7-2. Capping 로직 제거
**원인**: `updateStakeSlider`에서 `asset-hbt-display` 잔액을 읽어 슬라이더 값을 capping → 잔액이 101 HBT면 max가 101로 제한되어 범위가 6단계뿐

**수정**: `js/main.js` — capping 완전 제거, 잔액 검증은 `startChallenge30D`에서 처리

### 7-3. touch-action 설정
**수정**: `styles.css` — `.wallet-stake-slider`에 `touch-action: pan-y` 추가

### 7-4. 슬라이더 max를 보유 HBT로 즉시 제한
**수정**: `js/main.js` — `toggleChallengeSelection` 패널 열 때 현재 잔액으로 `updateChallengeSliderBounds` 즉시 호출
```js
const hbtText = document.getElementById('asset-hbt-display')?.textContent || '0';
const bal = parseFloat(hbtText.replace(/[^0-9.]/g, '')) || 0;
if (bal > 0 && window.updateChallengeSliderBounds) window.updateChallengeSliderBounds(bal);
```
