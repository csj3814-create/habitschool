## 🔄 HBT 전환 (Minting)

### 전환 비율 (난이도 조절에 따라 변동)
| 포인트 | HBT |
|--------|-----|
| 100P | 50~400 HBT |
| 200P | 100~800 HBT |
| 300P | 150~1,200 HBT |

- **변환 단위:** 100P 단위
- **일일 전환 한도:** 최대 5,000 HBT/일 (서버 제한)
- **온체인 유저 일일 한도:** 20,000 HBT/일
- **온체인 글로벌 일일 한도:** 500,000 HBT/일
- **전환 비율:** 매주 자동 난이도 조절 (최대 1P = 4 HBT)

### 전환 과정
```
습관 인증 → 포인트 적립 (Firebase)
→ 사용자가 전환 버튼 클릭
→ 서버에서 스마트 컨트랙트 mint() 호출
→ 사용자 지갑에 HBT 전송
→ 블록체인 탐색기(basescan.org)에서 tx 확인 가능
```

## 📉 Phase 기반 반감기 + 주간 난이도 조절

> *"일찍 시작할수록 유리하다"*  
> *"비트코인처럼 누적 채굴량 기반 반감, 주간 난이도 자동 조절"*

### 핵심 원리
- ❌ 시간 기반 → ✅ **누적 채굴 발행량 기반**
- 사용자 적으면 → 구간이 오래 유지 → **초기 참여자 혜택**
- 사용자 폭증하면 → 구간 빠르게 소진 → **인플레이션 자동 억제**

### Phase 구조

채굴 보상 풀: **70,000,000 HBT** (전체의 70%)

| 구간 | 채굴 풀 | 누적 경계 | 주간 목표 |
|------|-----------|-----------|----------|
| A (Phase 1) | 35,000,000 HBT | 35M | 140,000 HBT/주 |
| B (Phase 2) | 17,500,000 HBT | 52.5M | 70,000 HBT/주 |
| C (Phase 3) | 8,750,000 HBT | 61.25M | 35,000 HBT/주 |
| D+ (Phase 4~) | 8,750,000 HBT | 70M | 17,500~ HBT/주 |

> 채굴 풀 총합: 35M + 17.5M + 8.75M + 8.75M = **70,000,000 HBT**

### 주간 난이도 자동 조절

매주 월요일 0시(KST) 오프체인 스크립트가 새 비율을 계산하여 온체인에 반영:

| 조건 | 비율 변동 |
|------|----------|
| 채굴량 < 주간 목표 | 비율 ↑ (최대 2배/주) |
| 채굴량 > 주간 목표 | 비율 ↓ (최소 0.5배/주) |
| 절대 상한 | 4 HBT/P (100P = 400 HBT) |
| 절대 하한 | 0.01 HBT/P |

## 🔥 소각 (Burn)

### 1. 챌린지 실패 (달성률 80% 미만)
- 예치 HBT의 **50% 소각**, 나머지 50% 반환
- 예: 100 HBT 예치 → 80% 미만 시 50 HBT 소각, 50 HBT 반환

### 2. 외부 토큰 교환 수수료
- HBT → 외부 토큰 교환 시 교환량의 **2% 소각**
- 예: 10,000 HBT 교환 → 200 HBT 소각

### 소각 효과
- 총 공급량 감소 → 희소성 증가
- **반감기 + 소각 = 이중 디플레이션**
- 소각분은 복구 불가 (burn address 전송)

## 🏆 챌린지 스테이킹

슬라이더로 원하는 만큼 HBT를 예치하고 습관 인증에 도전!
모든 챌린지는 **식단+운동+마음** 3개 모두 기록해야 1일 인정.

| 티어 | 참가비 | 100% 달성 | 80%+ 달성 | 80% 미만 |
|------|--------|-----------|-----------|----------|
| ⚡ 3일 미니 | 무료 | 30P | — | — |
| 🔥 7일 위클리 | 50~5,000 HBT | 예치의 150% + 100P | 예치금 반환 + 100P | 50% 반환 / 50% 소각 |
| 🏆 30일 마스터 | 100~10,000 HBT | 예치의 300% + 500P | 예치금 반환 + 500P | 50% 반환 / 50% 소각 |

- **예치량**: 슬라이더로 자유 설정 (최소~최대 범위 내)
- **100% 달성**: 예치금 반환 + 보너스 (위클리 +50%, 마스터 +200%)
- **80% 이상**: 예치금 전액 반환 + 포인트
- **80% 미만**: 50% 소각, 50% 반환
- 보너스 HBT는 채굴 풀에서 별도 발행

> **3일 미니**: 3일 전부 완수해야 성공 (100% 필수)
> **80% 기준**: 7일 중 6일, 30일 중 24일 인증 완료

## 🔗 스마트 컨트랙트 구조

### 1. HaBit.sol — ERC-20 토큰 (v2)
- OpenZeppelin AccessControl (MINTER_ROLE) + ReentrancyGuard
- 최대 발행량 100,000,000 HBT 하드코딩
- 소수점 8자리 (decimals = 8)
- Phase 기반 반감기 (3,500만 / 5,250만 / 6,125만 경계)
- 주간 난이도 조절 (updateRate, smoothing 0.5x~2.0x)
- 서킷 브레이커: 글로벌 50만/일, 유저 2만/일
- 챌린지 스테이킹 (stakeForChallenge / resolveChallenge)
- 리저브 30M 컨스트럭터 프리민트

```python
def getConversionRate():
    # v2: 온체인 currentRate로 결정 (주간 조절)
    # Phase 경계: 35M / 52.5M / 61.25M
    minted = totalMintedFromMining
    if minted < 35_000_000: phase = 1  # A구간
    elif minted < 52_500_000: phase = 2  # B구간
    elif minted < 61_250_000: phase = 3  # C구간
    else: phase = 4  # D구간+
    # 실제 비율은 updateRate()로 주간 갱신
    return currentRate  # RATE_SCALE(10^8) 단위
```

### 2. HaBitStaking.sol — 챌린지 예치
- Ownable, 3 티어: mini(0), weekly(1), master(2)
- 100% 달성: 예치금 반환 + 보너스 (weekly +50%, master +200%)
- 80%+ 달성: 예치금만 반환
- 80% 미만: 50% 소각, 50% 반환
- 보너스는 운영자가 채굴 풀에서 별도 mint

### 3. HaBitExchange.sol — 외부 토큰 교환 (향후)
- HBT → 파트너 토큰 교환 또는 Uniswap V3 풀 활용
